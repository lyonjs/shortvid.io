Welcome to your Remotion project!

## Commands

**Install Dependencies**

```console
pnpm i
```

**Start Preview**

```console
pnpm start
```

**Render video**

```console
pnpm build
```

**Upgrade Remotion**

```console
pnpm run upgrade
```
